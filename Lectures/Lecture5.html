Introduction to CSS3

CSS helps separate how the page looks from the content. Some important dates:

  • 1996: CSS Level 1
  • 1998: CSS Level 2
  • CSS3: New features

CSS Syntax

Example:

selector - property - value

Like this: p color red

Ways to add CSS

  1. External file
  2. Inside style tag
  3. Directly in HTML tags

Colors

  • Names: Blue, Red
  • HEX: #ff0000
  • RGB: rgb(255,0,0)

Borders

You can add lines around things:

Example: 2px solid red

Why CSS?

It helps keep style separate from content so it is easy to change later.

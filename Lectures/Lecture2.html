Web Development Essentials

Professional Tools:

- Text Editors like VS Code and Sublime Text

- Browsers like Chrome and Firefox for testing

- Version Control tools like Git and GitHub

- Graphic Software such as Figma and Photoshop

HTML History Timeline

- 1991: HTML was first published

- 1999: HTML 4.01 became standard

- 2000: XHTML 1.0 released

- 2012: HTML5 was published

HTML5 Key Features

- Simple doctype declaration: !DOCTYPE html

- Multimedia support with audio and video tags

- Semantic elements like header and footer

- Canvas API for drawing graphics

- Improved form controls

Semantic Elements Examples

- header, nav, section, article

- aside, footer, figure, figcaption

Example of article structure:

article starts

header with h1 title

paragraph with content

footer with author name

article ends

Text Formatting Elements

- b for bold text

- strong for strong importance

- i for italic text

- em for emphasis

- mark for highlighted text

- del for deleted text

- sup for superscript

- sub for subscript

Special Characters you can use

-   means non-breaking space

- < means less than sign

- > means greater than sign

- © means copyright symbol

Basic HTML Document Structure

- !DOCTYPE html

- html tag with language attribute

- head section with meta charset and title

- body section with headings and paragraphs

Key concepts to remember

1. Elements are like building blocks, such as paragraph tags

2. Attributes add extra info to elements, like image sources

3. Nesting means putting tags inside each other properly

4. Semantics means using tags for their meaning, not just looks
